Welcome to the Sekiro Practice Tool!

To run the tool, double-click jdsd_sekiro_practice_tool.exe while the game is running.
To install the no-logo mod, copy `dinput8.dll` in the game's directory, next to `sekiro.exe`.

If you have issues or questions, thoroughly check the FAQ.
If you can't find what you need there, feel free to open an issue or join the discord.

You can find the FAQ here: https://github.com/veeenu/sekiro-practice-tool#readme
